## ğŸ’» Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:
- Installed Git.
- Installed Ruby
- Installed RSpec gem
- Create a local directory that you want to clone the repository.
- Open the command prompt in the created directory.
- On the terminal run this command git clone https://github.com/Salwa99/TDD-project.git



### Setup

Clone this repository to your desired folder:

```sh 
git clone https://github.com/Salwa99/TDD-project.git
  ```
  ```sh 
  cd TDD-project
  ```
  ```sh 
  bundle install
  ```
  ```sh
  gem install rspec
  ```
### Run Tests
  ```sh 
  rspec spec
  ```
